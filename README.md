# Wizard Arena

> A 2D top-down action survival game built with **Python** & **Pygame-CE**.

## Instructions

### Requirements

Install Pygame Community Edition:

```bash
python -m pip install pygame-ce
```

Run this in the Terminal of your IDE.

## Running the Application (IDE Method)

1. Open the project folder in your preferred Python IDE (e.g. VS Code, PyCharm)
2. Ensure Pygame is installed (see Requirements section above).
3. Open the `main.py` file in the editor.
4. Click the **Run** button in your IDE to start the game.

## How to Run (Terminal Method)

1. Download/extract the project files into a single folder.
2. The folder structure should looks like this:

```bash
project/
├─ abilities.py
├─ background.py
├─ coin.py
├─ enemy.py
├─ leaderboard.py
├─ main.py
├─ menu.py
├─ pause.py
├─ player.py
├─ resolution.py
├─ resource_manager.py
├─ settings.py
├─ shop.py
├─ tutorial.py
├─ wave.py
├─ assets/
│   ├─ fonts/
│   ├─ images/
│   ├─ sounds/
```

3. Before running the game, ensure your terminal’s working directory is the same folder that contains `main.py`.

4. Run the following in the Terminal of your IDE to launch the game:

```bash
python main.py
```

## Controls

- WASD → Move the wizard.
- Mouse → Aim the staff.
- Left Click → Cast an ability.
- Scroll Wheel → Swap abilities.
